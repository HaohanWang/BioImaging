Setup steps:
1. Install Thinkgear Connector and Neurosky Visualizer to your 32-bit computer
2. Plugin the device and make sure it works properly with Neurosky Visualizer
3. Install VLC player to the directory of C:/Program Files/VideoLAN/VLC. That means, the file of libvlc.dll should be under that path.
4. Clone from https://github.com/HaohanWang/BioImaging.git and import it as a Eclipse project.
5. You can start the program by running the MainFrame.java class
